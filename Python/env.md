## _错误行为_

（1）将 lambda 定义 放在 for 中【影响性能】

```python
def main():
    for i in range(10):
        filter1 = lambda x: x%2==0
        if filter1(i):
            print(i)

main()
```

（2）默认值设置为可变数据类型【结果不可控】

```python
def f(a, L=[]):
    L.append(a)
    return L

print(f(1)) # -> [1]
print(f(2)) # -> [1, 2]
print(f(3)) # -> [1, 2, 3]
```
