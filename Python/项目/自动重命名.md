方法二：使用正则匹配

```python
# 说明：只会将一级目录匹配到的重命名

import re
import os

def main(path):
    for file in os.listdir(path):
        if os.path.isfile(path+file):
            if new_name := re.findall("\][\S]*",file):
                os.rename(path+file, path+new_name[0][1:])
                
path = "D:/电影/"
main(path)
```
