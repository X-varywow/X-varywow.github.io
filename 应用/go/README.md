
Go 语言被设计成一门应用于搭载 Web 服务器，存储集群或类似用途的巨型中央服务器的系统编程语言。

对于高性能分布式系统领域而言，Go 语言无疑比大多数其它语言有着更高的开发效率。它提供了海量并行的支持，这对于服务端的开发而言是再好不过了。

>本文作为学过几种基础编程语言，学习Go语言的简要记录

GO 语音特性：
- 并发编程
- 内存回收
