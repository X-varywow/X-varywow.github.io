## 方案比对

几种方案：
- web
- python + web
- python + unity
- python + live2d


`unity 开发应用`

感觉 mediapipe js sdk 比 python sdk 简洁。。。

python + web 或 python + unity 收益都挺高的，目前还是 python + unity + socket 通信吧
