### 产品开发流程

①用户需求调研、市场调研，为产品定下一个基本的方向

②【可选】用户需求文档、用户体验地图，使用原型工具初步设计（墨刀等）

③开始产品开发，最小逻辑实现，基本框架实现；然后慢慢修改


### 小程序开发流程

①申请账号（主体：个人），安装开发者工具等。

②学习基础知识，框架知识

③两条实现路线：
- 从头开始
- 套用模板，改模板
