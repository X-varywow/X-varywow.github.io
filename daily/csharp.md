## hello world

C# 程序在 .NET 上运行；

.NET 是名为公共语言运行时 (CLR) 的虚执行系统和一组类库。 

CLR 是 Microsoft 对公共语言基础结构 (CLI) 国际标准的实现。 

CLI 是创建执行和开发环境的基础，语言和库可以在其中无缝地协同工作。
