## _grep匹配提取内容_

```bash
# 过滤出 包含 abc 或 123 的行，并显示行号
grep -n 'abc\|123' filename

# 过滤同时打印上一行
grep -B1 'key' filename

# 配合 cat 使用
cat ./docker-compose.yml | grep mnt

head a.py | grep import

# 排除包含 kube-system 的行
grep -v kube-system

# 使用正则匹配多个 ip(缩进严格) 
grep -E 'ip1|ip2|ip3'

# -i 会匹配所有大小写形式的 delete
grep "delete" /home/nacos/logs/nacos.log*
```

## _查看端口_

```bash
netstat -ano ::列出所有端口的使用情况

netstat -aon|findstr "8081" ::查看被占用端口对应的 PID

tasklist|findstr "9088" ::查看指定 PID 的进程

taskkill /T /F /PID 9088 ::结束进程
```


linux, mac查看端口占用，并关闭进程


```bash
ps -ef | grep postgres
lsof -i tcp:port
sudo kill -9 pid
```


命令含义：
- 'ps' Process Status
- -e 显示所有的进程
- -f 显示完整的格式 


输出含义：
- UID 用户ID
- PID 进程标识符
- TTY 关联的终端设备
- CMD 用于启动进程的命令
