

欢迎来到 非监督学习，主要用于处理无监督数据的学习问题。

<u>不要求事先标记好训练数据，通过学习数据的内在结构和模式来进行学习和预测。</u>


主要应用场景：
- 聚类分析(将样本划分为不同的组别)
- 异常检测
- 降维（减少描述数据所需要的值（或维度）的方法）
- 降噪（使用降噪算法来删除奇异值和缺失值）


</br>

主要算法：
- K-means 聚类
- DBSCAN
- 主成分分析 PCA
- 高斯混合模型 GMM
- t-SNE
